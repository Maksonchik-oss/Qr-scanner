# QR-SCANNER

## Описание программы

QR-SCANNER - программа для автоматического распознавания QR-кодов и штрихкодов из графических файлов с сохранением результатов в базу данных.

Программа позволяет быстро обрабатывать большое количество изображений, автоматически определять содержимое QR-кодов, сохранять историю сканирований и открывать найденные ссылки в браузере.

## Основные возможности

* Сканирование всех изображений из папки одной кнопкой.
* Распознавание QR-кодов.
* Распознавание штрихкодов.
* Поддержка форматов PNG, JPG, JPEG, BMP, TIFF и GIF.
* Сохранение результатов сканирования в базу данных SQLite.
* Автоматическое открытие найденных ссылок в браузере.
* Поддержка Telegram-ссылок.
* Поиск по сохранённым данным.
* Фильтрация результатов.
* Отображение статистики по найденным данным.

# Подготовка изображений

Перед началом работы необходимо создать папку:

```text
input_qr
```

Папка должна находиться рядом с файлом программы или блокнотом Jupyter.

Пример:

```text
project/
│
├── qr_scanner.py
├── input_qr/
│   ├── image1.png
│   ├── image2.jpg
│   └── image3.gif
```

Поместите в папку изображения с QR-кодами или штрихкодами.

# Работа с программой

## Запуск сканирования

Нажмите кнопку:

```text
SCAN FOLDER
```

Программа автоматически:

* найдёт все изображения в папке;
* выполнит распознавание QR-кодов;
* выполнит распознавание штрихкодов;
* сохранит результаты в базу данных;
* откроет найденные ссылки в браузере.

---

# Функции интерфейса

## SCAN FOLDER

Запускает сканирование изображений из папки.

## SEARCH

Поиск по:

* идентификатору записи;
* содержимому QR-кода;
* дате сканирования;
* времени сканирования.

## ALL

Показать все записи.

## TELEGRAM

Показать только Telegram-ссылки.

## URL

Показать только интернет-ссылки.

## TEXT

Показать только текстовые данные.

## CLEAR HISTORY

Удаляет всю историю сканирований.

## EXIT

Завершает работу программы.

---

# Статистика

В верхней части окна отображается статистика:

* TOTAL - общее количество записей;
* TELEGRAM - количество Telegram-ссылок;
* URL - количество интернет-ссылок;
* TEXT - количество текстовых записей.

---

# База данных

Для хранения результатов используется база данных SQLite.

Файл базы данных:

```text
qr_database.db
```

Структура таблицы:

| Поле      | Описание                 |
| --------- | ------------------------ |
| id        | Уникальный идентификатор |
| qr_text   | Содержимое QR-кода       |
| scan_date | Дата сканирования        |
| scan_time | Время сканирования       |

## Ошибка


## Ошибка



```text

```

```bash

```

## Ошибка

```text
Нет изображений в папке input_qr
```

Решение:

Создать папку `input_qr` и добавить в неё изображения для сканирования.

---

## Ошибка

```text
Окно программы не появляется
```

Возможные причины:

* код был запущен не полностью;
* отсутствуют необходимые библиотеки;
* возникла ошибка при запуске программы.

Решение:

1. Перезапустить Jupyter Notebook.
2. Проверить наличие всех библиотек.
3. Повторно выполнить запуск программы.

---

## Ошибка

```text
AttributeError: module 'cv2' has no attribute 'barcode'
```

Решение:

Обновить OpenCV:

```bash
pip uninstall opencv-python

pip install --upgrade opencv-python
```

# Ограничения программы

* Не поддерживается обработка PDF-файлов.
* Не поддерживается обработка видеофайлов.
* Не все типы штрихкодов могут быть распознаны корректно.
* Качество распознавания зависит от качества исходного изображения.
* Сильно повреждённые или размытые QR-коды могут не распознаваться.

---

# Безопасность

Программа может автоматически открывать ссылки, обнаруженные в QR-кодах.

Перед использованием рекомендуется убедиться, что изображения получены из надёжных источников.

Пользователь самостоятельно несёт ответственность за переход по найденным ссылкам.
